Guide to Jakarta JSON Processing TCK 2.0 Documentation

The Jakarta JSON Processing TCK 2.0 documentation includes the following:

  • The Jakarta JSON Processing TCK Release Notes provides the information on the test harness, the test suite and the platforms tested.
    • If needed, updated Release Notes are available at the Jakarta EE Specification web site for Jakarta JSON Processing: https://jakarta-ee/specifications/json-processing/2.0/
  • The Jakarta JSON Processing Technology Compatibility Kit, Version 2.0 User's Guide provides the information that you need to install, set up, and run the Jakarta JSON Processing TCK, Version 2.0. In addition, the guide provides the rules you must comply with to pass the Jakarta JSON Processing TCK.
  • The Javadoc Assertion List lists all the javadoc assertions that are tested by the Jakarta JSON Processing TCK.
