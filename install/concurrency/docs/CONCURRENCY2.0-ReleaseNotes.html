Jakarta Concurrency Technology Compatibility Kit, Version 2.0
Release Notes, September 2020

Kit Contents

The Jakarta Concurrency, Version 2.0 Technology Compatibility Kit (TCK) includes the following items:

  • Jakarta Concurrency TCK tests signature, API, and Spec tests:

    • A signature test that checks that all of the public APIs are supported in the Jakarta Concurrency Version 2.0 implementation that is being tested

    • API tests for all of the public APIs under the jakarta.enterprise.concurrent package

    • Spec tests for test assertions in the Jakarta Concurrency Utilities for Jakarta EE 2.0 Specification

Jakarta Concurrency TCK Facts

The test suite bundle contains the following:

  • 71 total tests
  • Signature Test

Platform Notes

The Jakarta Concurrency TCK tests have been built and tested with JavaTM Platform, Standard Edition 8 (Java SE 8).

The Jakarta Concurrency TCK tests have been run on the following platforms:

  • CentOS Linux 7

The Jakarta Concurrency TCK tests have been run against a Concurrency 2.0 Compatible Implementation.
