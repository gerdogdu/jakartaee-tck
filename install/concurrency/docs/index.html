Guide to Jakarta EE Concurrency TCK 2.0 Documentation

The Concurrency TCK 2.0 documentation includes the following:

  • The Concurrency TCK Release Notes provides the information on the test harness, the test suite and the platforms tested.
  • The Release Notes may be updated from time to time. Check the Eclipse Concurrency Specifciation Web Site: https://jakarta.ee/specifications/concurrency/2.0/.
  • The Concurrency Technology Compatibility Kit, Version 2.0 User's Guide provides the information that you need to install, set up, and run the Concurrency TCK, Version 2.0. In addition, the guide provides the rules you must comply with to pass the Concurrency TCK.
  • The Javadoc Assertion List lists all the javadoc assertions that are tested by the Concurrency TCK.
  • The Specification Assertion List lists all the specification assertions that are tested by the Concurrency TCK.
