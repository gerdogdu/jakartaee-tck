Guide to Jakarta Persistence TCK 3.0 Documentation

The Jakarta Persistence TCK 3.0 documentation includes the following:

  • The Jakarta Persistence TCK Release Notes provides the information on the test harness, the test suite and the platforms tested.
    • Updates to the Release Notes, if needed are available from the Jakarta Persistence Specification Web Site: https://jakarta.ee/specifications/persistence/3.0/
  • The Jakarta Persistence Technology Compatibility Kit, Version 3.0 User's Guide provides the information that you need to install, set up, and run the Jakarta Persistence TCK, Version 3.0. In addition, the guide provides the rules you must comply with to pass the Jakarta Persistence TCK.
  • The Javadoc Assertion List lists all the javadoc assertions that are tested by the Jakarta Persistence TCK.
  • The Specification Assertion List lists all the specification assertions that are tested by the Jakarta Persistence TCK.
