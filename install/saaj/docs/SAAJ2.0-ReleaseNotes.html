Jakarta SOAP Attachments Technology Compatibility Kit, Version 2.0
Release Notes, September 2020

Kit Contents

The Jakarta SOAP Attachments, Version 2.0 Technology Compatibility Kit (TCK) includes the following items:

  • Jakarta SOAP Attachments TCK tests signature, API, and end-to-end tests:

    • A signature test that checks that all of the public APIs are supported in the Jakarta SOAP Attachments Version 2.0 implementation that is being tested

    • API tests for all of the public APIs under the jakarta.xml.soap package

    • End-To-End tests that demonstrate sending/receiving SOAP messages to/from a URL endpoint as well tests for WSI compliance

Jakarta SOAP Attachments 2.0 TCK Facts

The test suite bundle contains the following:

  • 450 total tests

Platform Notes

The Jakarta SOAP Attachments TCK tests have been built and tested with the Apache Tomcat 8.5.x Web container running on the JavaTM Standard Edition 8 (Java SE 8) platform. If you are using a different Web server, you will need to make adjustments to the procedures outlined in this document.

The Jakarta SOAP Attachments TCK tests run on the following platforms:

  • CentOS 7

Installing, Setting Up, and Running the Jakarta SOAP Attachments TCK

Refer to the Jakarta SOAP Attachments TCK 2.0 User's Guide for complete instructions on installing, setting up, and running the Jakarta SOAP Attachments TCK. The online version of the JT Harness version 5.0 documentation is available here.
