Guide to JSON-B TCK 2.0 Documentation

The JSON-B TCK 2.0 documentation includes the following:

  • The JSON-B Release Notes provide the information that you need to install, set up, and run the JSON-B TCK, Version 1.0. The Release Notes are located in the TCK product bundle.

  • The JSON-B 2.0 Technology Compatibility Kit User's Guide provides the information that you need to install, set up, and run the JSON-B TCK as well as the rules you must comply with for certification.

  • The Javadoc Assertion List lists all the javadoc assertions that are tested by the JSON-B TCK.

  • The Specification Assertion List lists all the specification assertions that are tested by the JSON-B TCK.
