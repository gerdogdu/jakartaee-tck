Jakarta JSON Binding Technology Compatibility Kit, Version 2.0
Release Notes, September, 2020

Kit Contents

The Jakarta JSON Binding, Version 2.0 Technology Compatibility Kit (TCK) includes the following items:

  • Jakarta JSON Binding TCK tests signature, API, and end-to-end tests:

    • A signature test that checks that all of the public APIs are supported in the Jakarta JSON Binding Version 2.0 implementation that is being tested

    • API tests for all of the public APIs under the jakarta.json.bind package

    • Specification tests for the testable assertions required by the Jakarta JSON Binding 2.0 Specification.

Jakarta JSON Binding 2.0 TCK Facts

The test suite bundle contains the following:

  • 271 total tests

Platform Notes

The Jakarta JSON Binding TCK tests have been built and tested running on the JavaTM Standard Edition 8 (Java SE 8) platform.

The Jakarta JSON Binding TCK tests run on the following platforms:

  • Oracle Linux 7.1
  • Windows 10

Installing, Setting Up, and Running the Jakarta JSON Binding TCK

Refer to the Jakarta JSON Binding TCK 2.0 User's Guide for complete instructions on installing, setting up, and running the Jakarta JSON Binding TCK. The online version of the JT Harness version 5.0 documentation is available here.
